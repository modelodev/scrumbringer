# Story 4.4: Agenda v1 (deadlines personales)

## Status: Draft

## Story
**As a** member,
**I want** an agenda view of my upcoming task deadlines,
**so that** I can manage time commitments.

## Acceptance Criteria
1. **Due dates**: Tasks can have an optional `due_at`.
2. **My agenda endpoint**: `GET /api/v1/me/agenda` returns tasks with due dates in a time window.
3. **UI**: A simple agenda view exists (list or calendar).
4. **No surveillance**: Agenda is personal; admins see aggregates only.

## Tasks / Subtasks
- [ ] Extend task model to include `due_at` (optional)
- [ ] Implement `/api/v1/me/agenda` endpoint
- [ ] Implement UI agenda view
